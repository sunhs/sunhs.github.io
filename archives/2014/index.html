2014-11-15

somewhere only we know

音乐是生活的调剂，不仅因为它可以调动你的情绪，更在于它帮你把生活的碎片搜集起来，封装好，待你准备品味的时候，它会把当时的味道原原本本地还原给你。一刹那，你仿佛回到了一开始听这首歌的时空，你看到了当时的人，你感受到当时的空气。这是多么神奇的潘多拉盒子，藏着你的记忆。

我是个怀旧的人，我喜欢音乐就在于它总能让我回到过去，即便一曲结束，仅剩无限失落，但至少置身其中时，这个世界便与我无关了。

最近单曲循环的是keane乐队的somewhere only we know，一首充满感伤呐喊的英伦歌曲。
